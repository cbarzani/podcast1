## 📋 3 Simple Steps to Get Online

### Step 1: Connect chirokyan.net (Choose ONE option)

#### **Option A: Netlify (Recommended - Easiest)**
1. Go to **[netlify.com](https://netlify.com)**
2. Sign up (free account)
3. Drag and drop your website files
4. Add domain: `chirokyan.net`
5. Update DNS at your domain registrar:
   ```
   Nameservers:
   ns1.netlify.com
   ns2.netlify.com  
   ns3.netlify.com
   ns4.netlify.com
   ```

#### **Option B: Vercel**
1. Go to **[vercel.com](https://vercel.com)**
2. Sign up (free account)
3. Import project
4. Add domain: `chirokyan.net`
5. Update DNS:
   ```
   Type: A
   Name: @
   ```
